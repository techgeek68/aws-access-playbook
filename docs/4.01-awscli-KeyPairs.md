# AWS CLI - EC2 Key Pair Operations

## Create Key Pair (Linux/Unix)

Syntax:
```bash
aws ec2 create-key-pair \
  --key-name <KEY_PAIR_NAME> \
  --key-type rsa \
  --key-format pem \
  --query 'KeyMaterial' --output text > <KEY_FILE>.pem
```

> **Note:** `--key-type` accepts `rsa` (default) or `ed25519`. `--key-format` accepts `pem` (default) or `ppk` (for PuTTY). Both flags are optional if you want the defaults.

---

## Secure Private Key (Linux/Unix)

```bash
chmod 400 <KEY_FILE>.pem
```

---

## Create Key Pair (PowerShell)

Syntax:
```powershell
aws ec2 create-key-pair --key-name <KEY_PAIR_NAME> --key-type rsa --key-format pem --query "KeyMaterial" --output text > <KEY_FILE>.pem
```

Example:
```powershell
aws ec2 create-key-pair --key-name MyLoginKey --key-type rsa --key-format pem --query "KeyMaterial" --output text > MyLoginKey.pem
```

> **Note:** On Windows, both `aws` and `aws.exe` work with AWS CLI v2. Single quotes in `--query` must be replaced with double quotes in PowerShell.

---

## Secure Key (PowerShell)

Syntax:
```powershell
icacls <KEY_FILE>.pem /inheritance:r /grant:r "$($env:USERNAME):(R)"
```

Example:
```powershell
icacls MyLoginKey.pem /inheritance:r /grant:r "$($env:USERNAME):(R)"
```

---

## List Key Pairs

List names only:
```powershell
aws ec2 describe-key-pairs --query "KeyPairs[*].KeyName" --output text
```

List full details:
```powershell
aws ec2 describe-key-pairs --output table
```

---

## Delete Key Pair

Syntax:
```bash
aws ec2 delete-key-pair --key-name <KEY_PAIR_NAME>
```

Example:
```bash
aws ec2 delete-key-pair --key-name MyLoginKey
```
